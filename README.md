
# Prerequisites

* Node.js
* npm (node package manager)

# Install

First run `npm install` to install dependencies. You can then use `npm run dev`
to run locally for development. The command `npm run build` will build your
project. 

# Dataset

The example uses data from a study examining how gastric bypass surgery impacts over a thousand proteins.  

Yousri, N. A., Engelke, R., Sarwath, H., McKinlay, R. D., Simper, S. C., Adams, T. D., Schmidt, F., Suhre, K., & Hunt, S. C. (2022). Proteome-wide associations with short- and long-term weight loss and regain after Roux-en-Y gastric bypass surgery. Obesity, 30(1), 129–141. https://doi.org/https://doi.org/10.1002/oby.23303
